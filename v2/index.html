01
V1 vs V2 — What Changed and Why
Version 2 is not a downgrade. It is the smart way to build — launch what delivers value now, add complexity when revenue justifies it.
The PhilosophyEvery feature cut from V2 was cut for one reason — it requires significant time and cost but does not directly drive the first 1,000 users. The Event Calendar, Shop Finder, and Sponsor Directory do. Everything else can wait until the platform is making money.

Removed from V1

  • PSA / BGS / SGC population reports
  • Heritage / Goldin historical auction scrapers
  • Vertex AI live web search grounding
  • TikTok auto-posting
  • Instagram auto-posting
  • X (Twitter) auto-posting
  • Audio and video generation (ElevenLabs)
  • VIP press release monitoring
  • Full RAG pipeline with pgvector
  • 20+ site scraper coverage

Kept in V2

  • Event Calendar with interactive Google Map
  • Card Shop Finder with specialty filters
  • Sponsor Intelligence Directory
  • Full Admin Panel with analytics
  • User auth — save shows, set alerts
  • Basic AI chatbot (eBay sold data only)
  • LinkedIn auto-posting
  • Newsletter (casual monthly tier)
  • AI blog post generation
  • Email alerts via MailerLite

AI Chatbot — Lean Version

V2 Chatbot ScopeThe V2 chatbot answers questions using internal database data (shows, shops, sponsors) plus eBay sold listing data via the official eBay API. It does NOT use Vertex AI web search or Heritage/Goldin auction data in this version. It answers "shows near me", "shops in Houston", "eBay sold price for [card]". All other questions get a "I don't have that data yet — check [source link]" response.

Development Timeline — 6 to 8 Weeks

Foundation
Supabase DB schema, Vercel deploy, GitHub repo, auth system, admin panel skeleton. Leo gets a live staging URL on Day 1.

Google Maps integration, state zoom, city pins, click-to-drawer interaction, all event fields, mobile responsive.
Week 3
Event Calendar — Data + List View
List view with filters, TCDB and SCD scrapers, 5 priority shows manually entered. 50+ shows on the map.
Week 4
Sponsor Directory + Shop Finder
Sponsor category grid, basic profiles. Google Places shop seeding, map and list view, specialty tagging.
Week 5
User Auth + AI Chatbot
Registration, login, save shows and shops, email alerts via MailerLite. Basic Gemini chatbot with eBay API integration.
Week 6
Content Engine
Supabase webhook triggers, Gemini blog post generation, LinkedIn auto-post pipeline, monthly newsletter setup.
Week 7
Admin Panel + Analytics
Full admin dashboard, approve/reject workflows, analytics, user management, manual content trigger.
Week 8
QA + Launch
Cross-browser testing, mobile audit, performance check, admin walkthrough with Leo, DNS switch, production launch.
Weekly RhythmSunday call every week — 30-45 minutes. Live demo of that week's build. Email update before and after every call. Staging URL live from Day 1 so Leo can check progress anytime without waiting for a call.
